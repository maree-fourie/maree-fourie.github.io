1 Set theory

Expression Description
a ∈ A a is element of set A
∅ empty set
A ⊆ B A is subset of B. Every element of A is in B
A = B ⇔ A ⊆ B; B ⊆ A

1.1 Relations

A binary relation R on a set A is a set of ordered pairs of elements of A, that is, a subset of A × A.


R = {…, (ai, aj), …} : ai, aj ∈ A

R ⊆ A × A

Relation type Description
Reflexive ∀a ∈ A : (a, a) ∈ R
Symmetric (b, a) ∈ R → (a, b) ∈ R
Transitive (a, c) ∈ R → (a, b) ∈ R; (b, c) ∈ R
Equivalence R is Reflexive, Symmetric, and Transitive

1.2 Functions

A (1-ary) function on a set A is a binary relation F on A such that for every a∈A there is exactly one pair (a,b)∈F.


F : A → B  ⇒ ∀a ∈ A∃!(a, b) ∈ F

F(a) = b

1.3 Sets and formulas

Given formula φ(x, y1, …, yn) and sets A, B1, …, Bn. The set of all elements of A that satisfy the formula φ(x, B1, …, Bn) is {a ∈ A : φ(a, B1, …, Bn)}.


∅ = {a ∈ A : a ≠ a}

A = {a ∈ A : a = a}

A − B = {a ∈ A : a ∉ B}

A ∩ B = {a ∈ A : a ∈ B}
